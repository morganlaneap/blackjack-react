# Blackjack React

A simple blackjack game made for fun.

There are some base rules for this game:

- The dealer must draw to 16.
- The dealer must stand on 17 or higher.
- A win pays 2 to 1.
- Blackjack pays 3 to 2.
- Push returns your bet.

### Running it

```bash
yarn
yarn start
```
